## Objective

Installable package + pinned grammar + corpus fetch, so T1.1 can start with zero
setup.

## Deliverables

1. `pyproject.toml` — package `cobol_archaeologist` (src layout), Python 3.12;
   deps `tree_sitter==0.21.3`, `pydantic>=2`; extras `[dev]`: `pytest`, `ruff`.
2. Package skeleton exactly per the CLAUDE.md layout table — empty modules with
   one-line docstrings naming their owning task ID.
3. `vendor/tree-sitter-cobol/` — grammar commit
   `e99dbdc3d800d5fa2796476efd60af91f6b43d93` (only `src/parser.c`,
   `src/scanner.c`, `src/grammar.json`, `src/node-types.json`, plus LICENSE).
   Write the commit hash into `vendor/tree-sitter-cobol/PINNED`. Do not add the
   grammar as a git submodule — vendor the files (air-gapped deploy target).
4. `src/cobol_archaeologist/parser/_grammar.py` — `get_language() -> Language`:
   build the grammar from `vendor/` via `Language.build_library`, cache the
   `.so` under `build/` (gitignored), load and return. Idempotent; rebuilds only
   if `.so` missing.
5. `scripts/fetch_corpora.sh` — shallow-clone CardDemo at `59cc6c2fd7eb` and
   CBSA (`cicsdev/cics-banking-sample-application-cbsa`, record the commit you
   get in `data/corpora/PINNED`) into `data/corpora/`. Idempotent (skip if
   present).
6. `.gitignore` — `build/`, `data/`, `__pycache__`, `*.so`, `.pytest_cache`.
7. Move `src/cobol/spike_parser.py` → `docs/reference/spike_parser.py`
   (reference only, imports not required to work); delete `src/cobol/`.

## Gate (write first, in `tests/test_scaffold.py`)

- `import cobol_archaeologist` succeeds after `pip install -e .`
- `get_language()` returns a usable Language (parse the string
  `"       IDENTIFICATION DIVISION."` without exception)
- `vendor/tree-sitter-cobol/PINNED` content equals the hash above
- (skip-marked if `data/corpora/` absent) CardDemo checkout contains
  `app/cbl/CBTRN02C.cbl`

## Stop conditions

- The pinned grammar commit is unfetchable or its `src/` lacks `parser.c` →
  stop, report.
- `tree_sitter==0.21.3` won't install on the local Python → stop, report the
  version conflict; do NOT silently migrate to newer bindings (that's a
  deliberate decision for the chat).

## Out of scope

No preprocessor, no parsing logic beyond the gate's smoke parse — that is T1.1.
