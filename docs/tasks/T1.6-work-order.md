## D1 — `tools.py`: `RealToolLayer`
`RealToolLayer(corpus_root: Path, copybook_paths: list[Path], regulations=None)` implementing
`tool_types.ToolLayer` (assert `isinstance(self, ToolLayer)` in a test). Wiring:
- Parse-on-first-touch with an internal cache; call graph built once over the parsed set;
  `include_preamble=True` throughout.
- `read_paragraph` → parser spans + call_graph callers/callees; code text from **original
  source**, capped at `CODE_CAP_LINES` with `truncated=True` + full `ref`.
- `read_program` → ProgramView (names + spans only — no code dump).
- `find_callers`/`find_callees` → call_graph, NodeRef passthrough.
- `trace_variable`/`slice_on` → dataflow/slicer with A2 `program=` semantics.
- `resolve_copybook` → copybooks.expand, capped text + LineMap.
- `get_data_layout(record)` → NEW logic: walk the data-division AST for the record's field
  tree (name, level, PIC, REDEFINES, children). **VALUE text must be read from original
  source via LineMap, never preprocessed text** (see docs/preprocessor-notes.md — the
  continued-literal splice would corrupt D4 evidence).
- `grep(pattern)` → regex over original sources of the corpus, GrepResult capped at 200
  matches with `truncated=True`.
- `run_cobol(snippet, inputs)` → wrap model/run_cobol.run_cobol. A bare snippet (no
  IDENTIFICATION DIVISION) gets wrapped in a minimal batch program shell before compiling;
  a full program passes through unchanged. Compile failure → RunResult, never raise.
- `search_regulations(query)` → `raise NotImplementedError("Track C implements; see
  CONTRACT.md Part 1")` — typed stub per contract.

## D2 — `docs/tool-semantics.md` (consumer-facing, Track C will read this)
Enumerate every sentinel/edge semantic a ToolLayer consumer must know, accumulated across
T1.1–T1.5 reviews: `<preamble>` synthetic paragraph; `NodeRef.paragraph=""` = external
program entry (unparsed); `ref.program` may be a **copybook stem** for declaration sites;
`statement_kind` suffix `"?ambiguous"`; `"REDEFINES-alias"` marker sites; 88-level→parent
mapping; `truncated` + refetch convention; CICS `compiled_ok=False` = Tier-1 unavailable,
not an error; documented limitations (LINKAGE/COMMAREA, subscript flow, forest-root
entry_points pending F7/D6 split).
