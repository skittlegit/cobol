## Phase 1 [CODE] — Pin and gate the archive

1. `scripts/pin_regulations.py`: walk `data/regulations/sources/*.pdf`, write
   `data/regulations/sources/MANIFEST.json`:
   `{file, sha256, bytes, added: ISO-date, doc_role}` per entry (doc_role from
   the Phase-0 table). Idempotent; refuses to overwrite an existing hash with a
   different one (that's a provenance break — hard error).
2. `tests/test_sources.py`: for each MANIFEST entry, file exists + hash
   matches. `skipif` the sources directory is absent (CI without archives stays
   green), but the T2.5 gate itself requires it present.
3. Update the two source-URL notes in `data/manifest.json` regulations entries
   to point at `data/regulations/sources/MANIFEST.json`.
