## Model spec (frozen decisions — implement exactly)

All models: Pydantic v2 `BaseModel`, `model_config = ConfigDict(extra="forbid")`.

### `DriftType` (canonical literals — consumed by Track B mutate.py and eval)

```python
DriftType = Literal[
    "D1_stale_threshold",
    "D2_missing_rule",
    "D3_contradictory",
    "D4_stale_reference_data",
    "D5_boundary_error",
    "D6_dead_code",
    "D7_conformant",
]
```

### `CurrentValue`

- `kind: str` — e.g. `"duration_years"`, `"amount_inr"`, `"enum_set"`.
  Free string; the kind vocabulary is not frozen (Track B extends it in T2.1).
- `value: Any` (JSON-serializable).

### `RegulationClause`

- `doc: str` (required)
- `clause_id: str` (required)
- `version: str` (**required** — never optional; temporal axis is the novelty)
- `effective_date: datetime.date` (**required** — same rationale)
- `text: str` (required)
- `current_value: CurrentValue | None = None` — Optional because D2/D6
  clauses mandate a *check*, not a scalar. D1/D4/D5 instances should carry it
  (not schema-enforced in v1; enforced by the T2.3 generator).

### `CodeLocus`

- `programs: list[str]` (min length 1)
- `paragraphs: list[str]`
- `line_span: tuple[int, int]` — validator: both ≥ 1, `start <= end`
- `slice_vars: list[str]`
- `is_interprocedural: bool` (**required**, no default — it drives the
  headline stratification and must be a deliberate annotation)

### `Labels`

- `program_level: Literal["drift", "conformant"]`
- `paragraph_level: Literal["drift", "conformant"]`
- `line_level: list[int]` — empty list for conformant instances

### `Provenance`

- `source: Literal["synthetic", "real_curated"]`
- `base_program: str`
- `mutation: str | None = None` — required-in-practice for synthetic
  (validator below), absent for real_curated
- `annotator_notes: str | None = None` — for the real-curated seed (T2.5)

### `DriftInstance`

- `instance_id: str` — pattern `^drift_\d{6}$`
- `regulation_clause: RegulationClause`
- `code_locus: CodeLocus`
- `drift_type: DriftType`
- `labels: Labels`
- `gold_rationale: str` (required, non-empty)
- `provenance: Provenance`

**Cross-field validators on `DriftInstance` (model_validator, mode="after"):**

1. `drift_type == "D7_conformant"` ⇒ `labels.program_level == "conformant"`,
   `labels.paragraph_level == "conformant"`, and `labels.line_level == []`.
2. `drift_type != "D7_conformant"` ⇒ `labels.program_level == "drift"`.
3. `provenance.source == "synthetic"` and `drift_type != "D7_conformant"`
   ⇒ `provenance.mutation` is not None (a synthetic drift must record how it
   was made; MO-0 conformant instances may also record their benign edit but
   are not required to).

## Gate — `tests/test_schemas.py` (write first)

1. **Round-trip:** load fixture → `DriftInstance.model_validate_json` →
   `model_dump_json` → validate again → resulting model equals the first.
2. **Temporal fields unavoidable:** fixture with `effective_date` removed →
   `ValidationError`. Same for `version`.
3. **Literal enforcement:** `drift_type: "D8_bogus"` → `ValidationError`.
4. **Consistency validator:** D7 instance with `program_level: "drift"` →
   `ValidationError`; D1 instance with `program_level: "conformant"` →
   `ValidationError`.
5. **Line span:** `line_span: [161, 142]` → `ValidationError`.
6. **Extra fields forbidden:** unknown key at top level → `ValidationError`.
7. **Synthetic-needs-mutation:** synthetic D1 with `mutation: null` →
   `ValidationError`.
