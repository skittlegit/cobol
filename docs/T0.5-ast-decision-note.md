# T0.5 — AST Decision Note

**Decision: adopt tree-sitter (grammar: `yutaro-sakamoto/tree-sitter-cobol`) as the AST backend
for T1.1, fronted by a mandatory line-preserving preprocessing pass, with GnuCOBOL `cobc` retained
strictly as the compile/behavior oracle (T1.5, T2.2). Regex remains the fallback per the playbook.**

Date: 2026-07-02 · Task: T0.5 (Phase 0, M0) · Spike samples: AWS CardDemo `CBTRN02C.cbl` (731 L,
batch), `COSGN00C.cbl` (260 L, small CICS), `COACTUPC.cbl` (4,236 L, large CICS).

## The load-bearing finding: preprocessing is mandatory for _any_ backend

No candidate parses raw CardDemo CICS code — real mainframe compilers don't either (the CICS
translator and copybook expansion run first). The spike's preprocessor (line-count-preserving so
all coordinates map back to source):

1. **EXEC CICS/SQL/DLI → `CONTINUE`** masking, preserving the sentence-terminating period when
   `END-EXEC.` closed a sentence (this single period was the difference between 1 and 60
   paragraphs on `COACTUPC`).
2. **`COPY … REPLACING ==x== BY ==y==`** masking (pseudo-text breaks the grammar).

This pass belongs in `ingest/cleaner.py` regardless of parser choice — it is not a tree-sitter
workaround.

## Caveats carried into T1.1 (honest list)

- **Grammar maturity:** 38 stars, single maintainer. Vendor the grammar into our repo and **pin
  the commit hash** (also required by T7.2 determinism). Pin `tree_sitter==0.21.3` or migrate to
  the current bindings deliberately.
- **No `paragraph` container node:** the grammar emits `label`/`paragraph_header`; paragraph
  _spans_ must be computed as header→next-header. Deterministic, trivial, but it is our code.
- **COPY masking is spike-only.** T1.1 must do real copybook expansion (`parser/copybooks.py`)
  with a line-map to original coordinates — required anyway by MO-1× (stale threshold defined in
  a copybook).
- **EXEC block content is invisible to the AST.** T1.2's call graph needs a small side extractor
  over the masked spans for `EXEC CICS LINK/XCTL` edges (regex is adequate; the blocks are rigidly
  formatted).
- **Scope:** validated on 3 programs. T1.1's gate (10 hand-checked programs, nesting verified)
  remains open — this spike does not discharge it.
